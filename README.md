# 🕹️ Hangman Game (Python)

Welcome to the classic Hangman game built with Python!

## 📌 Features
- Word bank imported from `hangman_words.py`
- ASCII art stages and logo from `hangman_art.py`
- Tracks lives and guessed letters
- Easy-to-follow CLI interface

## 🎮 How to Play
1. Run `main.py` in any Python environment.
2. Guess letters to reveal the secret word.
3. You have 6 lives. Guess wrong and lose one.
4. Guess the word before your hangman is fully drawn!

## 📁 Files
- `main.py`: Main game logic
- `hangman_words.py`: Word list
- `hangman_art.py`: ASCII art for logo and stages
